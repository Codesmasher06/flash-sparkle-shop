# Flash Sale App

Flash Sale App is a web application where users can browse products, participate in flash sales, and make purchases in real-time. The app features dynamic inventory updates and a responsive user interface.

## Features
- Product listing with images and prices
- Flash sale countdown timer
- Add to cart functionality
- Responsive design for mobile and desktop

## Tech Stack
- HTML / CSS / JavaScript /node.js
- Hosted on Vercel
- GitHub repository for version control

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/codesmasher06/flash-sale-app.git
2. Open index.html in your browser
3. Enjoy the app!
